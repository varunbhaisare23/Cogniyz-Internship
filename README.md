# Cogniyz-Internship
This project involved a comprehensive analysis of a global restaurant dataset to uncover key insights into the food industry. Using Python, Pandas, and Matplotlib/Seaborn, I performed data cleaning, geographical mapping, and advanced statistical analysis to understand the factors that drive restaurant success and customer satisfaction.

Geospatial Analysis: Used Latitude and Longitude coordinates to map restaurant clusters and identify business "hotspots."
